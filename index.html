👋 Antes de empezar — 3 cosas importantes

  • 🎙 Permiso de micrófono: la primera vez que uses el dictado, tu navegador te pedirá permiso para usar el micrófono. Acéptalo para poder transcribir tu voz. El dictado funciona mejor en Google Chrome o Microsoft Edge.
  • 💾 Tus datos viven solo en este navegador: la lista de pacientes se guarda localmente en este equipo. Nada se envía a internet — pero si limpias los datos de navegación o cambias de computadora, la lista no se traslada sola. Usa el botón "Exportar respaldo" en la pestaña de Seguimiento para guardar una copia de seguridad y poder recuperarla después.
  • 🔒 Privacidad: esta herramienta no tiene servidores ni cuentas. La transcripción de voz la procesa el servicio de reconocimiento de tu navegador. Como con cualquier herramienta digital, evita incluir datos que identifiquen al paciente en el dictado si tu normativa local lo exige.
